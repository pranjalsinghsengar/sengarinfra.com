## Color Scheme

- **WhatsApp Green**: `#25D366` - Primary accent for WhatsApp elements
- **Primary Dark**: `#0F172A` - Headers, footer, text
- **Accent Purple**: `#7C3AED` - Secondary accents, gradients
- **White/Gray**: Backgrounds and contrast

## Key Interactive Elements

1. **Sticky Navbar**: Changes background on scroll
2. **Animated Stats**: Numbers count up when scrolled into view
3. **Hover Effects**: Cards lift on hover with shadow increase
4. **Floating Button**: WhatsApp button always accessible
5. **Mobile Menu**: Hamburger menu for mobile devices
6. **Smooth Scroll**: Anchor links scroll smoothly to sections

## Responsive Breakpoints

- **Mobile**: < 640px (Single column, hamburger menu)
- **Tablet**: 640px - 1024px (2 columns)
- **Desktop**: > 1024px (Full layout, 3-4 columns)
